Yes. The website should clearly mention guidance for both Central Government and Gujarat Government subsidy schemes.

14. Responsive Rules
Desktop: hero and form sections should use two-column layout.
Tablet: reduce card columns to two per row and keep enough spacing.
Mobile: stack hero, contact cards, form, consultation panel and map vertically.
Phone, WhatsApp and email links must be tap-friendly.
Keep sticky mobile CTA at bottom only if it does not cover form fields.
15. Developer Notes for HTML/CSS/JS Build
Use semantic HTML: main, section, form, address, footer.
Use tel: links for phone numbers and mailto: link for email.
Add front-end validation for name, mobile number, email and message.
Add a thank-you state after form submit. Since the site is HTML/CSS/JS only, connect the form later with Formspree, Netlify Forms, Google Sheet API, PHP handler or backend if required.
FAQ should be built with lightweight JavaScript accordion.
Use same header and footer component structure as other pages.
Keep image assets optimized and avoid heavy map iframe until final if speed matters.
